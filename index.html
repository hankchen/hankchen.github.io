使用pelican发布博客流程

1.激活pelican虚拟环境

activate pelican

2.切换工作目录

cd d:/temp/blog

3. 生成html

make html

4. 本地预览

make serve
可打开浏览器 localhost:8000进行预览

上传至git

初次提交

git init git add . git commit -m "first commit" git remote add origin https://github.com/xxx/xxx.github.io.git git push -u origin master

修改makefile简化发布流程

+修改github: publish下的代码： cd $(OUTPUTDIR) ; git add ./out ; git commit -m '更新博客' ; git push origin master +修改publish: $(PELICAN) $(INPUTDIR) -o $(OUTPUTDIR) -s $(PUBLISHCONF) $(PELICANOPTS) 为: `$(PELICAN) $(INPUTDIR) -o $(OUTPUTDIR) -s $(CONFFILE) $(PELICANOPTS)``

博客更新

make github
