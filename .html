使用pelican发布博客流程

1.激活pelican虚拟环境

activate pelican

2.切换工作目录

cd d:/temp/blog

3. 生成html

make html

4. 本地预览

make serve
可打开浏览器 localhost:8000进行预览

Proudly powered by Pelican, which takes great advantage of Python.

The theme is by Smashing Magazine, thanks!
